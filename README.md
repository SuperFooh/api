# api

## Status Codes que debemos manejar

[ ] - 200 --> todo ok (GET)
[ ] - 201 --> recurso creado (POST)
[ ] - 204 --> procesado sin retorno de info (DELETE)
[ ] - 400 --> BAD REQUEST, faltan datos, etc.
[ ] - 401 --> Falta de autenticacion
[ ] - 403 --> Autenticado pero no autorizado
[ ] - 404 --> No found
[ ] - 500 --> error de server
